# Signature email

Signature HTML pour Gmail regroupant 3 structures.

## Contenu du repo
- `signature.html` — fichier principal à copier dans Gmail
- `logo/makma_logo_transparent.png` — logo structure 1
- `logo/Flibusk.png` — logo structure 2
- `logo/Stephan_Boschat_logo_fond_transparent.png` — logo structure 3

## Comment utiliser
1. Ouvrir `signature.html` dans Chrome
2. Sélectionner tout `Cmd+A`
3. Copier `Cmd+C`
4. Gmail → Paramètres → Signature → Coller `Cmd+V`

## Comment modifier
Ouvrir `signature.html` dans VSCode et modifier les informations directement.
Les images sont hébergées sur GitHub et appelées via leur URL raw.
